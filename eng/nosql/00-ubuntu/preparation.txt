Some steps to follow after a fresh installation.

* Firefox
  "hamburger" icon -> Customize -> Toolbars
  + Menu Bar (enable)
  + Bookmarks Toolbar (enable)
  Done at the end

  Edit -> Preferences
    Downloads: always ask where to save files

* auto log in
  Google: Ubuntu 18.04 auto login
  https://linuxconfig.org/how-to-enable-automatic-login-on-ubuntu-18-04-bionic-beaver-linux
  Settings -> Details -> Users, unlock, set automatic login to "ON"

* install neovim

  $ sudo apt install neovim

* install mc

  $ sudo apt install mc

  Add the following line to the end of your ~/.bashrc file:

  alias mc='. /usr/lib/mc/mc-wrapper.sh'

* gedit

  Preferences -> View, display line numbers
              -> Editor, tab width: 4
                         check "insert spaces instead of tabs"

* install VSCode

  $ sudo snap install vscode --classic

  Recommended extensions:
  + Python
  + Python Extension Pack
  + Python for VSCode

* install htop

  $ sudo snap install htop

* useful aliases

  alias d='ls -al'
